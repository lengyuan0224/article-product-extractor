# article-product-extractor

> 「一切皆可Skill」SOLO 技能创作赛 参赛作品

## 🎯 作品介绍

**article-product-extractor** 是一个从文章链接中智能提炼产品与解决方案信息的 TRAE Skill。当用户提供文章链接，它能够自动按照标准化框架提取产品名称、核心功能、技术特点、应用场景、竞争优势等关键信息，结构化输出结果。

在信息爆炸的时代，我们每天都会读到大量技术文章、产品介绍、行业报告。人工整理这些信息费时费力，而这个技能能够帮你一键完成信息萃取，让你快速掌握产品核心要点。

## ✨ 核心功能

| 功能模块         | 功能描述                                                     |
| ---------------- | ------------------------------------------------------------ |
| 📥 **内容获取**   | 通过 WebFetch 自动获取链接文章内容，失败自动 fallback 到 WebSearch |
| 🔍 **多维度分析** | 从产品信息、解决方案、商业信息三个维度系统性提炼关键要素     |
| 📊 **结构化输出** | 输出格式清晰规范，包含产品概述、核心功能、技术特点、应用场景、竞争优势 |
| 📝 **质量检查**   | 内置完整性检查清单，确保信息不遗漏、准确可靠                 |
| 🎨 **灵活格式**   | 支持结构化摘要、对比表格、原文引用多种输出形式               |

## 🚀 安装使用

### 安装步骤

1. 将 `SKILL.md` 文件放置到 TRAE 项目的 `.trae/skills/article-product-extractor/` 目录下
2. 重启 TRAE 或刷新技能列表
3. 技能即可自动识别使用

### 使用方式

**直接调用：**

```
请使用 article-product-extractor 帮我分析这篇文章中的产品信息：https://example.com/article-url
```

**自动触发：**
当用户提供文章链接并要求"提炼产品信息"、"总结解决方案"时，TRAE 会自动识别并调用本技能。

### 示例

```
请帮我分析这篇文章中的产品信息：https://mp.weixin.qq.com/s/example
```

## 💡 创作思路

### 需求背景

在日常工作中，我经常需要阅读大量技术文章和产品介绍，每次都要手动整理产品信息，这个过程非常繁琐：

- 需要反复阅读才能抓住重点
- 整理的格式不统一，后续查阅不方便
- 多篇文章对比时，结构化整理工作量大

TRAE Skill 的理念就是"把专业经验固化成可复用的技能包"，于是我想到：能不能把"从文章提炼产品信息"这个高频需求标准化，做成一个 Skill？这样任何人遇到类似需求，都可以一键调用，获得结构化的结果。

### 设计思考

1. **为何选择这个场景？**
   - 高频刚需：几乎每个技术人、产品人都会遇到
   - 标准化程度高：提炼产品信息有固定的分析维度
   - AI擅长：大语言模型本身就具备理解和概括能力

2. **如何保证输出质量？**
   - 定义清晰的分析维度（产品/方案/商业）
   - 提供标准的输出模板
   - 增加质量检查清单环节
   - 区分事实和观点，标注不确定信息

3. **如何设计使用体验？**
   - 无需编程，纯 Markdown 定义，符合 Skill 规范
   - 自动触发 + 手动调用都支持
   - 输出格式灵活，支持多种展示方式

### 技术方案

整个技能基于 TRAE Skill 原生机制，仅需一个 `SKILL.md` 文件即可运行：

```
article-product-extractor/
└── SKILL.md          # 技能定义文档
```

**核心工作流：**

1. 获取文章内容 → 2. 多维度分析提炼 → 3. 按照模板结构化输出 → 4. 质量检查

依赖工具：

- `WebFetch` - 获取网页内容
- `WebSearch` - 备选获取方式
- `Read` - 支持本地文档

## ⭐ 创新点

1. **场景精准** - 聚焦"文章提炼产品信息"这个具体痛点，不贪大求全
2. **标准化框架** - 总结出产品信息+解决方案+商业信息三维分析框架，系统性不遗漏
3. **输出规范** - 提供多种输出模板，结果清晰易读，可直接用于报告
4. **质量意识** - 内置准确性、完整性、时效性检查，减少AI幻觉
5. **零代码** - 纯自然语言定义，无需任何代码，符合 TRAE Skill 理念

## 📊 实用性

这个技能解决了真实工作场景中的痛点：

- **节省时间**：原来需要10-30分钟的整理工作，现在一键完成，节省80%时间
- **提升质量**：标准化框架确保不遗漏关键信息，输出格式统一美观
- **可复用性**：一次创建，永久使用，任何需要分析产品信息的场景都能用
- **易于扩展**：用户可以根据自己的需求，轻松修改分析维度和输出模板
